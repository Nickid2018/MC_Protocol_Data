# Minecraft Protocol Data Repository

The repository contains the Minecraft Protocol Data for:
- Minecraft Protocol dissector for Wireshark ([Repo](https://github.com/Nickid2018/MC_Dissector))
- Protocol Viewer in Minecraft Wiki Interactive Tools ([Repo](https://github.com/mc-wiki/mcw-calc))

Some protocol data is from the [PrismarineJS/minecraft-data](https://github.com/PrismarineJS/minecraft-data) project.
